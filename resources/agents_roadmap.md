# LLM Agents: From Zero to One

LLM 代理在生成式 AI 領域中正獲得相當大的動力，因為它們可以處理反饋、維持記憶、為未來行動制定策略，並與各種工具合作以做出明智的決策。

如果你一直在尋找學習更多關於 LLM 代理的資訊，甚至想要建立自己的代理，那麼這個路線圖就是為你準備的！它充滿了很棒的免費資源，幫助你開始並隨時了解代理世界中發生的事情。
